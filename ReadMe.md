# Pet Match and Recognition System (Ongoing)

## Project Overview  

This project is designed to implement two neural networks with unique functionalities:  

1. **User-to-Pet Matching Neural Network:**  
   - Matches users to pet dogs based on their characteristics.  

2. **Pet Image Recognition Neural Network:**  
   - Utilizes Convolutional Neural Networks (CNNs) to recognize and classify pet species and breeds from images.  

Both functionalities will be accessible via a **Flask REST API**. Additionally, Google Cloud Platform (GCP) technologies may be integrated for enhanced scalability and performance.  

## Technologies Used  

- **PyTorch,Numpy,Pandas,Scikit-learn:** To build the models.  
- **Flask:** To provide a RESTful API interface for accessing the functionalities.  
- **Google Cloud Platform (GCP):** (Optional) For deployment, storage, and compute resources.  

## Goals  

- Deliver an engaging and accurate system to connect users with suitable pet dogs.  
- Build an efficient image recognition model to classify pet images with high accuracy.  
- Provide seamless access to these features through a robust API.
